# TETRA BlueStation Documentation

This repository contains the documentation for [TETRA-BlueStation](https://github.com/MidnightBlueLabs/tetra-bluestation), an open-source TETRA base station project.

The rendered and published documentation is available via the GitHub Wiki:

**(https://github.com/MidnightBlueLabs/tetra-bluestation-docs/wiki)**

All documentation changes should be made in this repository and submitted via pull request.  
The wiki is automatically updated from the main branch.

For contribution guidelines, issue reporting, and workflow details, please refer to the documentation itself.
